# Authentication API
| Route                | description         |
| -------------------- | ------------------- |
| POST /user           | create user         |
| PATCH /user/{userId} | update user         |
| POST /auth           | authenticate a user |
| PATCH /block         | block a user        |
| PATCH /unblock       | unblock a user      |
